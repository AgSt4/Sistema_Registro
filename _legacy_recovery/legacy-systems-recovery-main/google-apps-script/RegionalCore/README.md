# Regional Core (Standalone Logic)

**Tipo:** Librería / Script Autónomo
**Despliegue:** Se replica en las planillas regionales.

## ⚠️ Lógica de Activadores (Triggers)
Este sistema utiliza disparadores `onEdit` instalables/forzados para validar datos en tiempo real.

* **Mecanismo:** Detecta ediciones en celdas específicas y fuerza la validación.
* **Dependencias:** Requiere que el usuario tenga permisos de edición en los rangos protegidos para que el script se ejecute correctamente.
